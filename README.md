# Free-AI-For-All
**free-ai-for-all** is, a python library powered by **[pollinations.ai](https://pollinations.ai/)**
tis python library is super powerfull, if you know about **AI** and know what you are doing this can be alot, fun and powerfull, and also use this wisely.

### get started!

**type this in your terminal to, get started with coding with this library**
```bash
pip install free-ai-for-all
``` 
or if you want to, upgrade **type this**

```bash
pip install --upgrade free-ai-for-all
```

and then yyou just add this, at top of your python code

```python
import free_ai_for_all as fafa #this imports, the library as fafa short for, free ai for all
```
